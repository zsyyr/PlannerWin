Getting Started

Starting Planner

You can start Planner in the following ways:

Applications menu

Choose Office → Project Management.

Command line

To start Planner from a command line, type the following command, then press Return:

planner

there are no command line parameters at this time.

When You Start Planner

When you start Planner, the following window is displayed. As you can see, the Gantt Chart view is the default.

Figure 5. Planner Start Up Window


The Planner window contains the following elements:

Menubar.

The menus on the menubar contain all of the commands you need to work with files in Planner.

Toolbar.

The toolbar contains a subset of the commands that you can access from the menubar.

Statusbar.

The statusbar displays information about current Planner activity and contextual information about the menu items.

Editing Multiple Projects

More than one project may be open at the same time. Each project will have its own Planner window. Try it by pressing "CTRL-N" several times. You should get something similar to Figure 6. Several projects at the same time.
